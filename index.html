RoundArt

In this app, we will be making art with a circle and lines. Picture a clock, with numbers from 1 to 12. If we draw a line from each number to two times of the previous number (e.g. 1 joins to 2, 2 joins to 4 and so on, using modular arithmetic for larger numbers), it would look something like this:

The shape looks interesting. But what if we add more points instead of just 12? And what if we use different multiples to determine which points should be joined? Can we use fractions? What patterns can we create? The sandbox is all yours. (Hint: use the arrow keys for finer control of the sliders!)
